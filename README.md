# Lingo 🌍

App de idiomas feito em família, para chegar à fluência de verdade.

## Idiomas
- 🇬🇧 Inglês
- 🇮🇹 Italiano
- 🇫🇷 Francês
- 🇪🇸 Espanhol

## Como usar
Abra o arquivo `index.html` no navegador (Google Chrome de preferência).
Escolha o perfil da pessoa, o idioma e um tema. Estude com os cartões
(que falam em voz alta) e depois teste-se no quiz.

## Como funciona (para quem está aprendendo a programar)
- `index.html` → a estrutura da página
- `style.css` → as cores e o visual
- `app.js` → o conteúdo (palavras) e a lógica do app

Para adicionar novas palavras, edite a seção `CONTEUDO` dentro de `app.js`.

Feito com ❤️ pela família.
